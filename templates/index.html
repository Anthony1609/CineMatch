Project

About CineMatch

CineMatch is a full-stack web application built for CSC 309 that demonstrates User-Based Collaborative Filtering — the same technique used by Netflix, Spotify, and Amazon. Users rate films they've seen; the system finds similar users and predicts what else they'd enjoy using a weighted cosine similarity approach.

Tech Stack

Built With

Flask — Web framework & REST API routing
Pandas — User-Item Matrix via pivot_table()
NumPy — Cosine similarity via dot product & norm
HTML / CSS / JS — Responsive frontend with fetch API
Algorithm

How the Engine Works

  1. Load ratings.csv and build User-Item Matrix with Pandas
  2. Append the new user's ratings as a temporary row
  3. Compute cosine similarity between new user and all stored profiles
  4. Select the top-3 nearest neighbours by similarity score
  5. Predict ratings via similarity-weighted average
  6. Return top-6 highest predicted, unrated films
